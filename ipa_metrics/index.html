Readme

  この計算機は、IPAが公表している『ソフトウェア開発データ白書2014-2015』の「6.5 SLOC規模と工数」に記載されたデータを利用して、工数←→SLOCの変換を行うものです。

  ソフトウェア開発データ白書の注記にも記載のとおり、この計算式の結果は工数とSLOCの関係を特定のシステム開発事案について保証するものではありません。
